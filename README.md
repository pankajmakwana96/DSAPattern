## Quick Start

1. Clone the repository
2. Navigate to a pattern directory (e.g., `Two-Pointers/`)
3. Read the pattern README for an overview
4. Pick a problem and start coding!

```bash
# Example: Compile and run Java solution
javac ValidPalindrome.java
java ValidPalindrome

# Example: Compile and run C++ solution
g++ -std=c++17 ValidPalindrome.cpp -o ValidPalindrome
./ValidPalindrome
```

- C++ files use standard library (STL) and require C++11 or later
- Test cases are included in main() methods
- Solutions prioritize clarity and correctness over extreme optimization
